# Changelog

## 0.1.0 — 2026-04-10

Initial release.

- Three-layer architecture: Schema / Wiki / Raw
- Six operations: init, ingest, query, lint, trace, stats
- Three wiki templates: `--project`, `--research`, `--team`
- Shell utilities: `wiki-init.sh`, `wiki-lint.sh`, `wiki-stats.sh`
- Enrichment guide with five-dimension quality model
- Page templates for architecture, tech-stack, flow, entity, trace
- Example schemas for project and research wikis
